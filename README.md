# BitFinite Brand Kit

Official brand assets for **BitFinite (BFX)** — logos, wordmark, color palette,
and the **Miso** (cat) and **Ollie** (owl) mascots.

## Contents

| Path | What |
|---|---|
| `Lottie/` | Mascot animations (Lottie/dotLottie) — Miso & Ollie idle / rest / selected / loading, in light and dark, plus the `BitFinite-Duo.lottie` pair bundle |
| *(logos / wordmark / palette)* | BitFinite marks and the brand blue (`#0644f1`) |

The mascots map to the wallet's privacy modes: **Miso = Easy**, **Ollie = Incognito**.

## Usage

These are **brand assets, not open-source code.** You may use them to reference
or link to BitFinite, but not to imply affiliation, create derivatives, or
rebrand another project. See [LICENSE.md](LICENSE.md) for the full usage policy.

For anything beyond that, contact **bitfinitechain@proton.me**.
